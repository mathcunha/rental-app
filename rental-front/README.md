# rental-front

## Set ENV
* API_HOST=https://API_HOST
* API_URL=https://API_HOST/api/v1
* API_URI=api/v1
* GOOGLE_API_KEY=
* PIXABAY_KEY=
* JWT_SECRET=

## How to use

Download the example [or clone the repo](https://github.com/mui-org/material-ui):

Install it and run:

```sh
npm install
npm run dev
```
